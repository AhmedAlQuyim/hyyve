# Hyyve Company Site

Next.js 16 + Postgres marketing site for Hyyve. Phase 1 ships the public marketing site (Homepage, Solutions, About) reading content from the database.

## Local development

Prereqs: Node 18+, pnpm, Docker Desktop.

```bash
cp .env.example .env.local
docker compose up -d         # local Postgres on port 5433
pnpm install
pnpm db:migrate              # apply schema
pnpm db:seed                 # seed default site content (12 rows)
pnpm dev                     # http://localhost:3000
```

## Tests

```bash
pnpm test                    # vitest unit tests (content lib)
pnpm test:e2e                # playwright (boots dev server, ~16 tests)
```

## Database

- Local Postgres runs in Docker on host port 5433 (mapped to container 5432). The non-standard host port avoids conflicts with other Postgres instances.
- Schema lives in `lib/schema.ts`. Migrations are generated by `pnpm db:generate` and applied via `pnpm db:migrate`.
- Seed data lives in `db/seed.ts` and is idempotent (safe to re-run).

## Deployment (Railway)

1. Provision a Railway Postgres plugin → copy the `DATABASE_URL` it provides into the service environment.
2. Set the env vars from `.env.example`. Phase 1 only requires `DATABASE_URL`.
3. Build command: `pnpm build`
4. Start command: `pnpm db:migrate:deploy && pnpm db:seed && pnpm start`
   - Migrate is idempotent. Seed uses `onConflictDoUpdate` so re-running on each deploy is safe.

## Phases

- **Phase 1 (shipped):** Foundation + public site (Homepage, Solutions, About).
- **Phase 2 (planned):** Brief Wizard — chat-style modal that asks 5 discovery questions, calls OpenRouter to synthesize a scoped project brief, persists submissions to the `briefs` table, and notifies the team via Resend.
- **Phase 3 (planned):** Admin CMS — single-password login, edit any `site_content` field, revalidate the public pages on save.

See `docs/superpowers/specs/2026-04-26-hyyve-company-site-design.md` for the full spec and `docs/superpowers/plans/` for per-phase implementation plans.

## Tech stack

- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4 (design tokens via `@theme` in `app/globals.css`)
- Drizzle ORM + Postgres
- Vitest + Playwright

## Notes for contributors

- Windows users: the `start` script uses POSIX shell expansion (`${PORT:-3000}`). Use Git Bash or set `PORT` explicitly.
- The `hyyve-design/` directory contains the original prototype from Claude Design — kept in-repo as the canonical visual reference.
